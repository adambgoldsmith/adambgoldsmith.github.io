QuestQube

Express, GPT-3-Turbo

Built as a 5-week group project, QuestQube is a social web app with an emphasis on gaming and AI.

Save all of your favourite and most anticipated games to your profile and connect with new friends.

One of the core features of QuestQube is an interactive narrative adventure that will recommend a new video game based entirely on the user's choices. The recommender uses OpenAI's GPT-3-Turbo API to generate a unique and engaging story.

Nimbus Voyage

Python

Nimbus Voyage is a command-line, fantasy, airship RPG game created in Python. It was created as an assignment for class 1510 at BCIT CST during term 1.

Set sails in the open sky on your search for a lost companion. Nimbus Voyage features a simple navigation system which allows you to explore 3 different regions. An intutive combat and item system will test your strategy and wits.

Nimbus Voyage is the culmination of my Python learnings during my first semester at BCIT. Although it was only made in a few weeks, Nimbus Voyage packs in a ton of content and will even challenge experienced RPG enthusiasts.
